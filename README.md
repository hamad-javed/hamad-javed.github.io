# hamad-javed.github.io

Personal portfolio website — minimal, modern, black & white.

**Live:** [hamad-javed.github.io](https://hamad-javed.github.io)

## About

Portfolio site for Hamad Javed, a Full Stack Developer based in Lahore, Pakistan. Features selected projects, technical skills, education, and contact information.

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Google Fonts (Inter, JetBrains Mono)
- Font Awesome icons
- Hosted on GitHub Pages

## Sections

- **Hero** — Introduction with social links
- **Projects** — Catch-a-Jet, Boat Reservation & Kiosk, Regiekr8
- **Skills** — Frontend, Backend, Database & DevOps, Payments, State Management
- **Education** — BS Computer Science (UET Lahore), HackerRank Certification
- **Contact** — Email, location, social links

## Run Locally

Open `index.html` in a browser. No build step required.
